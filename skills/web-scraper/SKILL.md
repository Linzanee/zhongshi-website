# 网页内容抓取技能

## 功能概述
本技能用于抓取指定URL的网页内容，支持提取页面标题、正文文本、所有链接、图片等资源信息。

## 核心能力
- 获取网页原始HTML内容
- 提取页面标题和元数据
- 提取正文文本内容
- 提取所有链接（包括href和文本）
- 提取图片资源
- 处理相对路径转换为绝对路径

## 使用场景
- 数据采集和监控
- 内容分析和摘要生成
- 竞品信息收集
- 网页结构分析

## 操作步骤

### 1. 输入准备
- 用户提供目标URL
- 可选：指定需要提取的内容类型（标题、正文、链接、图片等）

### 2. 执行抓取
- 调用 `scripts/scrape.js` 脚本执行网页抓取
- 脚本使用 axios 获取页面内容
- 使用 cheerio 解析HTML结构

### 3. 输出结果
- 返回结构化的抓取结果
- 包含页面标题、正文、链接列表、图片列表等
